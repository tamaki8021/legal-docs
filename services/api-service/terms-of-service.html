API サービス利用規約

最終更新日: 2025年7月2日

第1条（適用範囲）

本利用規約（以下「本規約」）は、当社が提供するAPI サービス（以下「本API」）の利用条件を定めるものです。開発者（以下「利用者」）は、本規約に同意の上、本API をご利用ください。

第2条（定義）

本規約において、以下の用語は次の意味で使用されます：

  • API： Application Programming Interface の略称
  • API キー： 本API にアクセスするための認証情報
  • エンドユーザー： 利用者のアプリケーションを使用する最終利用者
  • レート制限： 一定時間内のAPI 呼び出し回数の上限
  • SLA： Service Level Agreement（サービス品質保証）

第3条（API の利用開始）

本API の利用を開始するには、以下の手続きが必要です：

  • 開発者アカウントの登録
  • 本規約への同意
  • API キーの取得
  • 必要に応じて利用プランの選択

当社は、申請内容を審査の上、利用可否を決定します。

第4条（API キーの管理）

利用者は、API キーについて以下を遵守する必要があります：

  • 第三者への提供・共有の禁止
  • 適切なセキュリティ対策による保護
  • 不正使用の疑いがある場合の即座の報告
  • 定期的な更新（推奨）
  • 利用終了時の適切な削除

第5条（利用制限）

本API の利用には、以下の制限があります：

  • レート制限： プランに応じた呼び出し回数上限
  • データ容量： 1回のリクエストあたりのデータサイズ上限
  • 同時接続数： 同時に維持できる接続数の上限
  • 利用目的： 規約で許可された用途のみ

制限を超過した場合、一時的な利用停止や追加料金が発生する場合があります。

第6条（禁止事項）

利用者は、本API の利用にあたり、以下の行為を禁止します：

  • 本API のリバースエンジニアリング
  • レート制限の回避行為
  • システムへの不正アクセス・攻撃
  • 他の利用者への妨害行為
  • 違法・有害なコンテンツの処理
  • スパム・フィッシング等の迷惑行為
  • 個人情報の不適切な取扱い
  • 著作権・知的財産権の侵害
  • API キーの不正利用・譲渡

第7条（サービス品質保証（SLA））

当社は、有料プランについて以下のSLA を提供します：

  • 可用性： 月間稼働率99.5%以上（無料プランは対象外）
  • レスポンス時間： 平均500ms以下
  • サポート対応： 営業日24時間以内の初回回答

SLA 未達の場合、利用料金の一部返金または無料利用枠の提供を行います。

第8条（料金・支払い）

有料プランの料金については、以下の通りです：

  • 料金は利用プランページに記載の通り
  • 月末締め・翌月請求
  • 支払い方法：クレジットカード、銀行振込
  • 未払い時は利用停止の場合あり
  • 料金変更は30日前に通知

第9条（データの取扱い）

本API を通じて処理されるデータについて：

  • 利用者がデータの管理責任を負う
  • 当社は処理に必要な範囲でのみデータを保持
  • バックアップは暗号化して保存
  • 法的要請時を除き第三者に提供しない
  • 利用者の要請により適切に削除

第10条（知的財産権）

本API およびドキュメントに関する知的財産権は当社に帰属します。利用者は、以下を遵守する必要があります：

  • 当社の商標・ロゴの無断使用禁止
  • API の複製・再配布の禁止
  • ドキュメントの改変・二次配布の禁止
  • ライセンス条件の遵守

第11条（免責事項）

当社は、以下について責任を負いません：

  • 利用者のアプリケーションの不具合
  • 第三者サービスとの連携問題
  • インターネット接続の不安定
  • API を利用したサービスの内容
  • エンドユーザーとの紛争
  • 機会損失・間接損害

第12条（サービスの変更・終了）

当社は、以下の場合にサービスを変更・終了できます：

  • 機能追加・改善： 事前通知（1週間前）
  • 仕様変更： 事前通知（1ヶ月前）
  • サービス終了： 事前通知（6ヶ月前）
  • 緊急時： 事前通知なし

第13条（利用停止・契約解除）

以下の場合、利用停止または契約解除を行うことがあります：

  • 本規約への重大な違反
  • 料金の長期滞納
  • 虚偽情報による登録
  • 反社会的勢力との関係判明
  • その他当社が不適切と判断した場合

第14条（サポート）

技術サポートは以下の方法で提供します：

  • 開発者ポータルでのドキュメント提供
  • FAQ・チュートリアルの提供
  • メール・チケットシステムでの問い合わせ対応
  • 有料プランではより高度なサポート提供

第15条（規約の変更）

本規約は、法令の変更やサービスの改善に伴い変更される場合があります。重要な変更については、30日前に利用者に通知いたします。

第16条（準拠法・管轄裁判所）

本規約は日本法に準拠し、本API に関する紛争については、当社の本店所在地を管轄する裁判所を専属的合意管轄とします。
